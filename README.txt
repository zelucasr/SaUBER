# SaUBER

Desenvolvimento do projeto da disciplina de WEB II

Descrição:
Assim como no UBER, o SaUBER trabalha com cadastro de profissionais (no UBER são os motoristas, no SaUBER são médicos das diversas áreas, enfermeiros, pronto socorristas, etc), e a demanda dos clientes.
Caso precise de um atendimento ou esteja passando por alguma emergência, o usuário cliente tem acesso aos profissionais mais próximos a sua localização, facilitando assim seu atendimento.
